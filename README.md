 🚀 Installation

Clone the repository

```bash
git clone https://github.com/SMHO179/local-chat.git
```

Go into the project

```bash
cd local-chat
```

Install dependencies

```bash
pip install -r requirements.txt
```

▶️ Run

Start the server

```bash
python cli/server.py
```

Start the client

```bash
python cli/client.py
```
